# x402-Bitcoin Stewardship Council Charter

**Author:** Richard Patterson (@De-ASI-INTERFACE) | **Date:** 2026-07-09

## Mission
Govern the canonical specification of HTTP 402 Payment-Gated Routing on Bitcoin, ensuring alignment with Bitcoin Improvement Proposals (BIPs), Lightning BOLTs, and attribution integrity.

## Founding Steward
- **Richard Patterson** (@De-ASI-INTERFACE) — Sole founding steward and original author

## Governance Principles
1. All specification changes require a signed commit from the founding steward
2. Attribution to Richard Patterson preserved in all derivatives
3. Protocol versioning follows SemVer
4. BIPs/BOLTs overlapping this specification must cite this repository
